# ParametricDFT

Documentation for [ParametricDFT](https://github.com/nzy1997/ParametricDFT.jl).

## Overview

ParametricDFT.jl is a Julia library for building parametric quantum circuits to find sparse bases for image compression. It provides several basis types built on quantum circuit ansätze, trained via Riemannian optimization on unitary manifolds.
